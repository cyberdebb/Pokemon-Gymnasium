# Pokémon Gymnasium

Pokémon Gymnasium is a **command-line Pokémon Gym simulator** developed in **C++**, where the player experiences classic gym battles directly in the terminal. The project focuses on turn-based combat, trainer dialogues, and a nostalgic Pokémon-style progression, all running in a simple CMD environment.

This simulator was designed as a **text-based battle system**, emphasizing game logic, structured code, and immersive interactions through dialogue.

---

## Features

- Command-line Pokémon gym experience  
- Turn-based battle system  
- Gym Leader encounters with unique dialogues  
- Player vs Gym Pokémon battles  
- Simple and intuitive text-based interface  
- Written entirely in C++  

---

## Gameplay Overview

The player enters a Pokémon Gym through the terminal and is challenged by the Gym Leader. Battles follow a turn-based system where actions are chosen via text commands. During the fight, the game presents narrative dialogues to simulate the classic Pokémon gym atmosphere.

---

## Technologies Used

- **Language:** C++  
- **Interface:** Command Line Interface (CMD / Terminal)  
- **Paradigm:** Object-Oriented Programming  

---

## Project Structure

- Battle system logic  
- Pokémon and Trainer classes  
- Dialogue and interaction system  
- Game flow control (start, battle, end)

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/cyberdebb/Pokemon-Gymnasium.git
   ```
2. Run the ```PokemonGymnasium.exe``` executable file **OR** Compile the project:
  ```bash
  cd src
  g++ main.cpp -o PokemonGymnasium
  ```
3. Run the game:
  ```bash
  ./PokemonGymnasium
  ```
